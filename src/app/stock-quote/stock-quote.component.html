Stock Quote Component

This component demonstrates using retry and error handling with Angular's Http service.

The retry(n) operator tells the Observable that if it errors, it should call the producer function n more times to try to get value(s) out of it. So retry(2) will attempt between 1 and 3 http requests before giving up and calling error on the observers.

The catch operator receives whatever value was passed to its error observer, and returns an Observable that will produce a value that will fail gracefully. In this case, the observable just returns an object with an error message.
